# `pyxis-defs`

A monorepo of definitions for applications written in the [Pyxis memory-structure DSL](https://github.com/ferrobrew/pyxis).

Organised by application and version. JSON documentation is automatically generated, to be viewed in [the viewer](https://ferrobrew.github.io/pyxis/).

For the Pyxis language reference, see [docs/language.md](https://github.com/ferrobrew/pyxis/blob/main/docs/language.md). For backend-specific docs (Rust codegen, JSON schema, C++ ABI), see the [docs directory](https://github.com/ferrobrew/pyxis/tree/main/docs).
